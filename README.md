# IdentityV
版本介绍:<br>
两个版本的人类按键完全相同<br>
Both版本使用右键攻击，Space,Alt为主要技能键，人屠共用左键翻越障碍;Swapper版本使用左键攻击，Space,Alt为主要技能键，需要手动将翻越障碍键改成F当你要玩屠夫的时候<br>
如何使用:<br>
首先确保第五人格没有在运行<br>
在:\你的第五人格文件夹\dwrg2\Documents 中找到pc_key_setting_data.json并删除它<br>
将你下载的json文件放到此文件夹并删除pc前面的括号和括号内的单词<br>
如果你找不到你的第五人格文件夹，右键在你桌面的第五人格，点击打开文件位置<br>
如何便捷切换：<br>
在确认按键已经更改之后请点击下方的上传云端<br>
如果是Swapper版本:在玩屠夫之前，将翻越障碍键设置为F；在玩人类之前，点击云端下载让翻越障碍键重新设置为鼠标左键<br>
